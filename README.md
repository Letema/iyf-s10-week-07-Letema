# Week 7: JavaScript Best Practices

## 📌 Overview
This project focuses on improving JavaScript skills by adding data persistence, state management, and clean code practices.

## 🚀 Features
- To-Do List with localStorage (add, delete, complete)
- Notes stored in localStorage
- Form auto-save using sessionStorage
- Theme toggle (light/dark mode)
- Shopping cart with quantity and total
- Data persists after page refresh

## 🧠 Concepts
- localStorage & sessionStorage
- JSON.stringify / JSON.parse
- State management
- Code organization (modular structure)
- Clean code practices
